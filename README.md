ORM UI for DVelum Core 4.x
======

## Installation


```
composer require --dev dvelum/orm-ui
php ./console.php /extension-add/dvelum/orm-ui
```


## Launch
Start PHP cli-server ```php ./vendor/bin/orm-ui```

Open: [http://localhost:8000](http://localhost:8000)
